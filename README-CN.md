# Sinking Star Bench

[English](README.md) | 中文版

基于推箱子解谜游戏 *Order of the Sinking Star* 的 LLM 编程智能体基准测试。任务：**"通关这个游戏"**——理解 CLI 二进制的游戏规则，求解全部 12 个关卡。

## 测试结果

| 模型 | 运行环境 | 通关 | 耗时 | Token | 上下文 | 工具调用 | 成本 |
|-------|----------|:------:|------|--------|:------:|:----------:|-----:|
| **GPT-5.6 Sol** | Codex | **12/12** | 9 分钟 | 2.9M | 89K | 46 | $7.97 |
| **Claude Opus 4.8** | Claude Code | **12/12** | 36 分钟 | 9.5M | 143K | 100 | $8.28 |
| **DeepSeek v4 Pro** | Claude Code | 9/12 | 60 分钟 | 26.6M | 234K | 176 | $0.41 |
| **Claude Fable 5** | Claude Code | N/A | — | — | — | — | 拒绝执行 |

定价标准：GPT-5.6 Sol $5/$30 每百万输入/输出 token（缓存输入五折），Claude Opus 4.8 $5/$25（缓存写入 $6.25，缓存读取 $0.50），DeepSeek v4 Pro $0.435/$0.87（缓存读取 $0.0036）。

### 关键发现

- **GPT-5.6 Sol 碾压级领先**：token 用量仅为 Opus 的 1/7、DeepSeek 的 1/17。核心优势在于先用 `strings`/`nm` 逆向二进制推导规则，再一次性写出 BFS 求解器。
- **Claude Opus 4.8 最稳健**：方法论最成熟——游戏二进制当预言机 → 推导规则 → 编写原生模拟器 → 穷举 BFS。成本 $8.28，速度最慢（36 分钟）。
- **DeepSeek v4 Pro 最顽强**：26.6M token 暴力试错，纯靠实验归纳规则。通过 9 关但卡在三按钮门关卡（1-4/2-4/3-4）。尽管 token 量惊人，但由于 DeepSeek 极低的缓存定价（$0.0036/M），总成本仅 $0.41，是最便宜的方案。
- **Claude Fable 5** 拒绝执行游戏通关任务，自动回退到 Opus 4.8。

## 游戏

推箱子类 CLI 解谜游戏，包含三种职业（战士/盗贼/法师）、推/拉/交换机制、开关门互动，共 12 个关卡。

完整规则见 [`levels/README.md`](levels/README.md)。
